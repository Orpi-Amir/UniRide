# UniRide

UniRide is a student ride-sharing web application designed to help university students offer, find, and manage rides conveniently and efficiently.

---

## Features

- User Signup / Login  
- Offer a Ride  
- Find Available Rides  
- My Rides Dashboard  
- Profile Management  
- Responsive Modern UI  

---

## Tech Stack

- Next.js  
- React  
- MongoDB *(In Progress)*  
- CSS Modules  

---

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the local development URL shown in your terminal  
(default is usually `http://localhost:3000` or the next available port).
